# Codeveda-Frontend-internship-level 3
# 🚀 Frontend Development Projects

This repository contains two frontend projects demonstrating modern UI development, reusable components, and advanced animations.

---

## 📌 Project Overview

This project includes:

1. Component-Based Web Application (Task Manager)
2. Advanced Animations & Interactive UI
3. Portfolio Contact Form with Email Integration

---

## 🧩 Task 1: Component-Based Web Application

A simple task management app built using HTML, CSS, and JavaScript.

### ✨ Features
- User Login & Registration (using localStorage)
- Add / Delete Tasks
- Dashboard UI
- Reusable Components (Navbar, Card, Button)
- Accessibility (ARIA attributes)
- Component Documentation Page

### 🧠 Concepts Used
- Component-based structure
- State management using localStorage
- Clean UI design

---

## 🎬 Task 2: Advanced Animations

Interactive UI animations using GSAP.

### ✨ Features
- Page Load Animations
- GSAP Timeline Animations
- Scroll-triggered Effects
- Parallax Scrolling
- Hover Effects
- Magnetic Button & Ripple Effect

### 🧠 Concepts Used
- Animation timelines
- Scroll-based triggers
- Smooth transitions

---

## 📧 Contact Form (Email Integration)

A working contact form integrated using EmailJS.

### ✨ Features
- Real-time email sending (no backend)
- Form validation
- Loading indicator
- Success / Error messages

---

## 🛠️ Tech Stack

- EmailJS
